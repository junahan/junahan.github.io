Emacs YASnippet 备忘录

本文是简单记述 yasnippet 配置和基本使用的备忘录。

1 Snippet 基本配置

  • 执行 M+x package-install 即可通过 MELPA 安装 YASnippet
  • 开启全局配置:
1
2

(require 'yasnippet)
(yas-global-mode 1)
  • 开启 per buffer 模式:
1
2

(yas-reload-all)
(add-hook 'prog-mode-hook #'yas-minor-mode)

2 安装 yasnippet-snippets 系列模版

yasnippet-snippets 是由 AndreaCrotti 维护的一系列模版。可以通过执行 M+x package-install 命令安装它。该安装包会更新 yas-snippet-dir 以自动加载该系列模版。也可以安装其他系列模版并通过配置 yas-snippet-dir 来加载模版。

默认加载模版目录配置 yas-snippet-dir 如下：

(“～/.emacs.d/snippets” yasnippet-snippets-dir)

3 定制 Snippet

通过运行命令 M+x yas-new-snippet 命令创建一个新的模版。如下是我用于在文章中插入摘要标签的模版：

1
2
3
4
5
6
7
8

# -*- mode: snippet -*-
# name: abs
# key: <abs
# --

#+BEGIN_abstract $1
  $0
#+END_abstract

将如上文件保存至默认 snippet 目录 (~/.eamcs.d/snippets) 即可。
